# Plantilla Base Para Sufeo.ia

Usa esta plantilla para generar páginas SEO consistentes para Clinvetia.

## Objetivo

- Keyword principal: `{{keyword_principal}}`
- Keyword secundaria: `{{keyword_secundaria}}`
- Intención: `{{informacional|comercial|transaccional}}`
- URL objetivo: `{{slug}}`

## Estructura recomendada

### Title

`{{keyword_principal}} | Clinvetia`

### Meta description

`Descubre cómo {{keyword_principal}} ayuda a clínicas veterinarias a {{beneficio_principal}} con Clinvetia.`

### H1

`{{keyword_principal}} para clínicas veterinarias`

### Intro

- Explica el problema principal en 2-3 frases.
- Introduce a Clinvetia como solución.
- Incluye la keyword principal una vez y una variante semántica.

### H2 Problema

- Describe el cuello de botella operativo o comercial.
- Añade señales concretas: tiempo, pérdidas, carga de recepción, citas perdidas.

### H2 Cómo lo resuelve Clinvetia

- Atención automatizada
- Triaje o clasificación
- Agenda o recordatorios
- Seguimiento y medición

### H2 Casos de uso

- Caso 1: `{{caso_1}}`
- Caso 2: `{{caso_2}}`
- Caso 3: `{{caso_3}}`

### H2 Beneficios

- Beneficio operativo
- Beneficio comercial
- Beneficio para el propietario

### H2 FAQ

- Incluye 4-6 preguntas reales relacionadas con la keyword.
- Respuestas breves y concretas.

### CTA final

- CTA principal: `Reservar demo`
- CTA secundaria: `Calcular ROI`

## Reglas editoriales

- Usar siempre `Clinvetia`, nunca `ClinvetIA`.
- No escribir texto genérico de IA; aterrizar siempre al contexto veterinario.
- Priorizar lenguaje comercial claro sobre claims vagos.
- Incluir enlaces internos a `/solucion`, `/como-funciona`, `/escenarios`, `/faqs`, `/demo` o `/contacto` cuando tengan sentido.
- Evitar duplicar títulos o intros entre páginas.

## Schema recomendado

- `BreadcrumbList`
- `FAQPage` si hay preguntas frecuentes
- `Service` si la página es comercial

## Enlaces internos mínimos

- 2 enlaces a páginas de intención comercial
- 2 enlaces a páginas de apoyo informacional
